Escape do Tsunami — Jogo para Gravação Entrega 2026-04-14 · Desenvolvido para Athos (YouTuber)

1. Como abrir o jogo

Passo a passo:
1. Abra o Roblox Studio
2. Clique em File → Open from File
3. Selecione o arquivo game_ready.rbxl desta pasta
4. Clique no botão ▶ Play (testar sozinho) ou ▶ Start Server (testar com múltiplos players)
Não use o arquivo default.project.json diretamente — ele é para desenvolvimento com Rojo. Para gravar, use sempre o game_ready.rbxl.

3. Fluxo recomendado para gravação

Roteiro sugerido:

1. Iniciar na área principal → tsunami passando → sobreviver
2. F8 → Teleport: SECRET SHOP → mostrar a loja de pulos
3. Comprar/dar pulos na ordem: James → JJ → Mana → Pdoro → Matheus → Caylus → Athos
4. Cada pulo → mostrar velocidade aumentada, partículas, efeitos visuais
5. F8 → Teleport: YOUR BASE → mostrar pedestais com brainrots coletados
6. Voltar para área principal → tsunami com Athos (velocidade máxima)

4. Mecânicas do jogo

Brainrots (itens coletáveis)

  • Aparecem automaticamente na área principal
  • Caminhe sobre eles para coletar
  • Cada brainrot gera renda por segundo ($2/s até $999M/s)
  • Na base, ficam expostos nos pedestais 3D
  • No painel "YOUR BASE" (canto direito), botão Sell para vender

Jump Shop (Loja de Pulos)

  • Localizada na Secret Shop
  • Entre pela parede rachada no fundo da loja
  • 7 pulos de Youtubers, cada um com custo e efeito diferente
  • Para gravar sem cumprir condições → use F8 → Give: [pulo]

Tsunami

  • Onda varre a área principal a cada intervalo
  • Fique em plataformas altas para sobreviver
  • Pulo JJ dá escudo azul que protege da onda
  • Para disparar manualmente → F8 → Wave

GalaxyBat (Pulo Matheus)

  • Aparece no Backpack após dar o pulo Matheus
  • Equipe o item → clique perto de um Noob → Noob é lançado para longe
